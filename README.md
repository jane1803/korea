# 아리아드네 랜딩페이지 (Next.js)

## 실행 방법

이 프로젝트는 **Node.js가 필요**합니다.

1) Node 설치 확인

```bash
node -v
```

2) 의존성 설치

```bash
cd "ariadne-landing"
npm install
```

3) 개발 서버 실행

```bash
npm run dev
```

브라우저에서 `http://localhost:3000` 를 여세요.

## 커스터마이징

- 메인 콘텐츠: `src/app/page.tsx`
- 전역 스타일: `src/app/globals.css`
- 간단 UI 컴포넌트: `src/components/ui.tsx`
